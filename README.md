## Installation & Setup
### Backend Setup
1. Navigate to the `Backend/` folder:  
   ```sh
   cd Backend
   ```
2. Install dependencies:  
   ```sh
   npm install
   ```
3. Start the backend server:  
   ```sh
   node index.js
   ```

### Frontend Setup
1. Navigate to the `frontend/` folder:  
   ```sh
   cd frontend
   ```
2. Install dependencies:  
   ```sh
   npm install
   ```
3. Start the frontend development server:  
   ```sh
   npm run dev
   ```
